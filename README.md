# Конфиг для Neovim 0.12+

Использует встроенные автодополнение и пакетный менеджер `vim.pack`, доступные с Neovim 0.12. Используется встроенный **lsp**.

- Настройки, подключаемые в `init.lua`, и вспомогательные функции:

  ```sh
  lua
  ├── config
  │   ├── autocmds.lua # Общие автокоманды, например, восстановление курсора при входе
  │   ├── keymaps.lua  # Общие сочетания клавиш
  │   ├── options.lua  # Основные настройки редактора
  │   └── theme.lua    # Автозагрузка и сохранение темы, выбранной через colorscheme, в theme.json
  └── utils.lua        # Вспомогательные функции
  ```

- `plugin/*.lua` — плагины и их настройки. Плагины устанавливаются и подключаются через `vim.pack.add`.

- `after/lsp/*.lua` — настройки языковых серверов, которые расширяют дефолтные, те что идут с `nvim-lspconfig`. Чтобы полностью предопределить настройки, их нужно класть в `lsp/`.

- Используй `<leader>?`, чтобы посмотреть все пользовательские сочетания через `whickey`.

- Для управления языковыми серверами и другими средствами разработки используй `:Mason`. Предустановленные языковые сервера перечислены в `ensure_installed` файла `plugin/lsp.lua`.

- Подсветка синтаксиса, фолдинг и пр. операции с кодом осуществлются через `tree-sitter` (встроенная поддержка). См. `TSInstall` (установка новых парсеров), `TSUpdate` и пр.
